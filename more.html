Code for Democracy

Code for Democracy is a nonpartisan nonprofit organization dedicated to creating tools to allow users to explore connections between campaign contributions, political narratives, and legislative outcomes.

I created a map visualization tool for the organization that allows users to visualize the organization's datasets by geographic bounds; this can be viewed here.Overall, I gained technical experience with Flask, Vue, and more extensive use of the Pandas library; more importantly, though, I got the chance to take a dive into real-world software engineering and gain experience with the development process.

Additionally, I am currently developing a topic modeling prototype to enable users to explore the relationships between current topic points in the news and on social media -- stay tuned!
